Repository Metrics

All SEAgle metrics related to Repository and development processes(Repository Based Metrics)

Category: REPO_METRICS

Mnemonic:
COM_PROJ

 Number of Commits of a Project

Metrics related to repository (such as commit count).

Mnemonic: COM_COUNT_VER

 Number of Commits per Version

Count the number of commits from a previous version. This will walk back the current version until it encounter\nthe previous version. Therefore for a given project it is important which versions are required for analysis. If intermediate versions are skipped this metric will result in a greater number of commits per version.
