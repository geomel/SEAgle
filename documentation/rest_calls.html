Seagle REST services are available through:

http://java.uom.gr:8080/seagle2/rs/{service_path}

GET POST PUT

GET

Get Project Name

URI

/project/name?purl=projectUrl

Parameters

projectUrl:https git path url

Description

Returns the name of the project with the given url

Response

200 OK
202 Accepted – If the required process has been scheduled successfully
404 Not Found
500 Internal Server Error

Get Project Version

URI

/project/versions?purl=projectUrl

Parameters

projectUrl:https git path url

Description

Returns a list with version IDs, version names and version dates of the project with the given path in json format

Response

200 OK
202 Accepted – If the required process has been scheduled successfully
404 Not Found
500 Internal Server Error

Get Project Registered Metrics

Response

200 OK
202 Accepted – If the required process has been scheduled successfully
404 Not Found
500 Internal Server Error

Get Metric values for the whole history of a project

URI

/metric/single/values/project/?purl=projectUrl&metricMnemonic=MNEM

Parameters

projectUrl:https git path url

metricMnemonic:The mnemonic of the desired metric

Description

Returns a list with the value of the specified metric for this project for all analyzed versions.

Response

200 OK
202 Accepted – If the required process has been scheduled successfully
404 Not Found
500 Internal Server Error

Get Metric values specific version of a project

URI

/metric/single/values/project/forVersion?purl=projectUrl&metricMnemonic=MNEM&versionID=commitVersionHashID

Parameters

projectUrl:https git path url

metricMnemonic:The mnemonic of the desired metric

versionID:The hash id from the commit of the desired version

Description

Returns a list with the value of the specified metric for this project for the specified version.

Response

200 OK
202 Accepted – If the required process has been scheduled successfully
404 Not Found
500 Internal Server Error

Get Project smell summary

URI

/smells/summary

/smells/summary/{projectName}

Parameters

Path parameter {projectName}

Description

 
												Returns a summary with the smells of all projects of the given project. Example: 
{ 
      "metrics": [], 
      "name": "v0.4.12", 
      "smells": [ 
        { 
          "className": "project version", 
          "quantity": 5, 
          "smellName": "God Class" 
        }, 
        { 
          "className": "project version", 
          "quantity": 11, 
          "smellName": "Data Class" 
        }, 
        { 
          "className": "project version", 
          "quantity": 0,

    },
