Source Code Metrics

SEAgle metrics related to source code(Source Code Metrics) like LOC, LCOM and CBO.

Category: Source Code Metrics

Mnemonic:
WOC

 Weight of Class

Weight of Class is the number of non-accessor methods in a class divided by the total number of members of the interface

Mnemonic: WMC

 Weighted Method Complexity (Count)

The WMC metric is the sum of the complexities of all class methods. It is an indicator of how much effort is required to develop and maintain a particular class.

Mnemonic: TCC

 Tight class cohesion

TCC is defined as the relative number of directly connected methods. Two methods are directly connected if they access a common instance variable of the class

Mnemonic: NOPA

 Number of public attributes

NOPA is the number of non-inherited attributes that belong to the public interface of a class

Mnemonic: NOF

 Number of Fields

Number of Fields of a class

Mnemonic: NOM

 Number of Methods

Number of Methods of a class

Mnemonic: NOAM

 Number of Accessor Methods

The number of the non-inherited accessor-methods declared in the interface of a class

Mnemonic: LOC

 Lines of Code

Lines of Code for a Class without blank lines or comments

Mnemonic: LCOM2

 Lack of Cohesion in Methods 2 (LCOM2)

LCOM2 is the percentage of methods that do not access a specific attribute averaged over all attributes in the class. If the number of methods or attributes is zero, LCOM2 is undefined and displayed as zeroDetailed specification at:http://www.aivosto.com/project/help/pm-oo-cohesion.html

Mnemonic: CBO

 Coupling Between Objects

The coupling between object classes (CBO) metric represents the number of classes coupled to a given class (efferent couplings, Ce). This coupling can occur through method calls, field accesses, inheritance, arguments, return types, and exceptions.

Mnemonic: ATFD

 Access to foreign data

ATFD represents the number of external classes from which a given class accesses attributes, directly or via accessor-methods. The higher the ATFD value for a class, the higher the probability that the class is or is about to become a god-class.ATFD is equivalent to the Out Degree of the corresponding node
